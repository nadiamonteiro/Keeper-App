# Keeper-App

<h2>About The Project</h2><hr>
<img src="https://user-images.githubusercontent.com/59107225/121367607-1154b080-c958-11eb-9160-a6c5bef2ecd0.png" alt="Blank app template">
This project is a note-taking application that can be used to create notes and delete them.
  
<h3>Built With</h3><hr>
<p>This project was created using <a href="https://reactjs.org/">React.js</a> on <a href="https://codesandbox.io/">CodeSandbox</a>.
